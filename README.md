# scinference
This R package implements the inference methods for synthetic and counterfactual controls proposed by Chernozhukov et al. (2020a,b). The papers are available here: https://arxiv.org/abs/1712.09089 https://arxiv.org/abs/1812.10820
# Installation
```
install.packages("remotes")
library(remotes) 
remotes::install_github("kwuthrich/scinference")
library(scinference)
```
